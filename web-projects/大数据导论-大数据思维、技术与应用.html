大数据导论-大数据思维、技术与应用

数据导论-大数据思维、技术与应用

作者：王名涵

选题简介

数字经济在国民经济中的比重越来越大，大数据已经成为重要的战略性资源，政府也提出了包含大数据中心在内的“新基建”概念。

作者介绍

本书作者为何明、何红悦、罗玲、禹明刚、徐兵、刘云。本作的作者们为中国人民解放军陆军工程大学教授何明所领导的调研小组，故对何明教授进行详细介绍

在本书中，何明教授以其在军队学科领域的卓越成就和丰富经验，对大数据进行了深入的研究和总结。作为一名中国人民解放军陆军工程大学的教授，何明教授不仅在军事领域有着丰富的专业知识和经验，还在学术界取得了显著的成就，是军队学科拔尖人才、科技英才，备受重视。

在书中，何明教授从自己的专业角度出发，对大数据进行了全面系统的介绍和分析。他的教学背景和科研经历使得他对大数据的解读更加深入，内容更为专业。他以“高、新、特”三个亮点为本书编写的特色，不仅内容详尽，专业度高，而且非常适合作为高校教材，也可作为管理人员学习大数据的培训资料。

何明教授希望通过本书的学习，读者们能够成为大数据的受益者。他对读者们的期望和祝愿体现了他对大数据事业的热忱和信心。作为读者和学习者，我们也应该借此机会，充分利用何明教授的知识和经验，不断提升自己的大数据能力，为中韩两国的数字经济和国防军事的发展做出贡献。

总的来说，何明教授的专业背景和深厚造诣为本书的内容提供了坚实的支撑，使得本书成为一部颇具权威性和实用性的大数据著作。通过对本书内容的总结，读者们可以更好地理解大数据的重要性和应用前景，从而更好地应对未来的挑战和机遇。

第一章、大数据发展历史

数据大爆炸

电子领域的摩尔定律的核心内容为集成电路（Integrated Circuit,IC）上可容纳的晶体管数目大约每两年增加一倍。在大数据时代，数据生成量每两年增加一倍。

小数据到大数据数据的转变

大数据发展为战略物资

数据资源成为一种新型的战略性资源，不少国家已经将大数据上升为国家意志，在战略层面进行整体筹划布置。 国外战略: “棱镜门” 事件清楚地警示人们，数字主权早已成为国家之间博弈的空间，失败的代价是任何国家都难以承受的。 美国: 美国的大数据建设与应用始于政府数据开放。2009年，为方便民众使用各类政府数据、提高政府数据的透明度，美国设立了Data.gov网站，开放了政府和企业收集的海量数据，涵盖大约50个细分门类。 美国政府发布了《联邦大数据研发战略计划》。 欧盟: 如高水平科研机构林立、顶尖人才众多、硬件基础设施相对完善等，但也面临数据层级复杂、各国数据难以共享等难题。2015年1月，欧盟大数据价值联盟正式发布了《欧盟大数据价值战略研究和创新议程》。 国内战略: 既面临前所未有的历史机遇，也面临难以预测的风险和挑战。因此，只有迎难而上，把握时机，直面挑战，强化多要素、多部门、各领域协同联动，才能推动我国大数据建设又好又快发展。

发展规划: 《大数据产业发展规划 (2016—2020年)》。

要成为数据强国不能一蹴而就，观念和思维方式的转变更不是一朝一夕的事情。因此，我国要打牢大数据发展的根基，真正让“云端”的理念落到实践。

数据制度坏境

数据参与分配需要法律制度的保障。我国应建立健全数据这一无形资产相关的财产界定、审计核准、评估制度，建立有力的数据监管机制，建立健全个人隐私信息保护和数据产权保护制度，推动数据的有效治理。

 

第二章节、大数据的应用

  • 金融交易: 主要应用于高频交易领域，通过大数据算法自动执行交易决策。

  • 商品营销: 企业通过挖掘和分析用户需求，提供个性化服务，例如电子商务平台通过用户的浏览记录来预测用户对商品购买的潜在需求。

  • 公共安全: 政府基于大数据技术提供数字化手段，为反恐维稳、预防犯罪与各类案件侦破提供支持，例如根据外卖信息抓捕犯罪分子。

  • 医疗健康: 大数据可以辅助流行病预测、疫情防控、个人健康管理，例如新冠肺炎疫情期间的智能诊断技术。

  • 城市管理: 基于大数据做出最优化的决策，例如救护车实时获取道路拥堵信息选择通行时间最短的路线。

第三章节、大数据的主要技术框架

MapReduce

通过将应用程序分解为许多并行计算指令，运行速度快。

Hadoop

Hadoop 是一个可进行批处理的开源大数据处理框架，其充分利用集群优势进行高速计算和存储。用户无须了解 Hadoop 的细节即可直接开发其大数据应用程序。

Hadoop 实现了 MapReduce 编程模式，即将应用程序分解为许多并行计算指令，这些指令能在集群的任意节点上运行，并以分布式文件系统 (Distributed File System,DFS) 来存储所有计算节点的数据，从而极大地提高了整个集群的带宽。

Spark

是一款高效的海量数据处理通用计算引擎，运行速度比Hadoop快100倍以上。Spark 是一款开源的海量数据处理通用计算引擎。Spark 比 Hadoop 的 MapReduce 更高效。Spark 的有向无环图 (Directed Acyclic Graph,DAG) 执行引擎可以支持循环数据流和内存计算，运行速度比 Hadoop 快 100 倍以上，其基于磁盘的运行速度也比 Hadoop 快 10 倍以上。Spark 支持多种编程语言，如 Scala、Java、Python 和 R 语言，简单、易上手，便于各领域使用

第四章、大数据安全体系

大数据安全技术：

大数据的安全体系架构依旧还在不断完善。

  1. 敏感数据识别技术：敏感数据识别技术是一种用于识别和标记敏感信息的技术，可以帮助机构有效管理和保护敏感数据。例如，一家银行可以使用敏感数据识别技术来自动检测客户账户中的身份证号码、信用卡号码等敏感信息，以确保这些信息不会被泄露或滥用。

  2. 数据防泄露技术：数据防泄露技术是一种用于保护数据不被未经授权的访问或泄露的技术。例如，企业可以采用数据加密、访问控制和安全审计等技术来防止员工意外或恶意泄露敏感信息。

  3. 密文计算技术：密文计算技术是一种在数据仍然加密的情况下进行计算和处理的技术。例如，云计算服务提供商可以使用密文计算技术来确保客户上传的数据在云端进行计算时仍然保持加密状态，从而保护数据的隐私和安全。

  4. 数字水印技术：数字水印技术是一种在数字内容中嵌入隐藏信息的技术，用于确认数据的来源和完整性。例如，在数字图像中嵌入数字水印可以帮助确定图像的版权信息和来源，防止未经授权的复制和使用。

  5. 区块链技术：区块链技术是一种去中心化的分布式数据库技术，可以确保数据的安全和不可篡改性。例如，医疗行业可以使用区块链技术来建立安全的健康数据交换平台，确保患者的敏感医疗数据在医疗机构之间安全地共享和传输。

区块链的安全挑战：

复杂结构与缺乏安全评估方法： 区块链结构复杂，缺乏系统级安全评估方法。

网络安全技术的无效化： 传统的网络安全技术无法有效应用于区块链构建的对等网络。

量子计算对现有加密技术的威胁： 随着量子计算技术的发展，现有的哈希函数、公钥加密算法等技术变得不再安全。

数据脱敏是数据使用安全管理的核心技术。拥有特权账号的用户是最易将敏感数据外泄的人员，因此对拥有特权账号的用户需要进行动态实时的数据脱敏，以防止用户敏感数据的泄露。

过程中的数据安全-数据脱敏流程：

  1. 数据脱敏的三个步骤：

    • 抽取：从生产库中抽取需要脱敏的敏感数据，并根据系统配置的脱敏规则进行匹配。

    • 漂白：进行一系列的隐私变形操作，确保未脱敏的敏感数据进行正常的产出和存储行为。

    • 发布：将脱敏之后的数据发布至测试库中，提供给相关人员使用。

  2. 数据脱敏处理：

    数据脱敏流程根据核心生产库的业务场景和数据的安全级别，采用常见的方法如替换法、重排法、加密法、截断法、掩码法等。

不同场景下的脱敏技术实现：

静态脱敏技术：一般用于非生产环境中，解决测试库、开发库所用的生产库数据量与数据间的关联问题，以及排查问题或进行数据分析。

动态脱敏技术：一般用于生产环境中进行实时脱敏。动态脱敏相对于静态脱敏更有难度，需要在确保业务正常运行的前提下，做到数据的动态脱敏。

数据脱敏与使用安全：

数据脱敏与使用安全密切相关，大数据系统的核心是DFS和MapReduce引擎，设计语言、API的多样性拓展了安全审计的覆盖面，也加大了数据解析的难度。

第五章、大数据的实际应用场景

军事大数据

指挥员可以利用功能强大的移动设备，如专用智能手机，向处于战术边缘的部队发送信息。一些最新的技术，例如谷歌眼镜，可以作为增强现实的应用平台，使战士们拥有穿戴式显示器，在移动中获取各种信息。

大数据下的指挥控制新特点：

  1. 作战决策科学化：

    • 大数据技术的发展使得指挥层级变得更少，简化了指挥体系结构，从“树状”向“网状”转变。

    • 决策权的分配更加科学，重新制定了作战过程中的各项重大决策，促使联合作战的程度不断提高。

  2. 行动控制协同化：

    • 智慧军营大数据的概念模型有助于实现行动控制的协同化。

智慧军营大数据的概念模型：

医疗大数据

区块链加医疗大有可为：

  1. 数据共享：区块链技术支持医疗机构间在数据平台上实现数据共享，使得医学影像检索、建模等更为便捷，有助于医务工作者进行诊疗和健康管理。

  2. 试点项目：阿里健康与常州市合作推进了“医联体 + 区块链”试点项目，旨在探索区块链技术在医疗联合体底层技术架构中的应用，促进医疗机构间的数据互操作。

  3. 安全保障：区块链技术能够保证医疗健康大数据在存储和传输过程中的安全性和可靠性，降低了数据泄露和篡改风险，提高了数据安全性。

  4. 优势：区块链技术在医疗健康领域有着独特的优势，包括数据的厚积薄发、应用价值的独特性以及未来医疗联盟链的发展前景。

 

社会大数据

社会治理：

社会治理大数据是指利用大数据技术和方法，对社会管理中产生的大量数据进行采集、整合、分析和应用，以促进社会管理现代化和提升治理能力。

公共安全：

公共安全大数据的目标包括预测预警智能化、预案体系数字化、信息资源共享化、指挥控制可视化、风险防控网格化等。例如，在新冠肺炎疫情暴发后，利用警务大数据平台实现了对人、车、物的精准摸排，为疫情精准防控提供了支持。

市场监管：

市场监管大数据体系旨在建设一个囊括市场监管所有过程及产品生命周期的数据链。通过数据收集与集成、数据处理和分析、数据管理等环节，实现市场监管的精准化监管及智能化服务。其中，物联网、互联网和区块链等技术被广泛运用于市场监管，以加强食品安全监管、净化市场环境、防范交易纠纷等。

农业大数据

  1. 共享休闲农业：将大数据与共享经济融合，诞生了共享休闲农业，为消费者提供休闲农业体验。大数据在共享休闲农业中发挥作用，通过深度挖掘游客的旅游线路和消费偏好，帮助农民为发展乡村旅游进行选址。

  2. 农产品质量追溯：大数据在农产品质量追溯中起到重要作用。通过标签数据和终端查询数据，实现对农产品生产过程的追溯，包括基础数据、质量管控数据、生产履历数据等，以及追溯二维码信息，帮助消费者了解农产品的生产过程。

  3. 农业环境监测：大数据用于监测农业环境，包括农田定点监测气象数据和病虫害监测数据，通过分析预测和发布预警信息，实时提供给数据管理系统，帮助农民做出农业生产决策。

  4. 农作物病虫害防治：大数据在农作物病虫害防治中发挥作用，通过监测预警数据和历史数据，结合数据分析与建模，实施精准营销，评估防治效果，提高农作物病虫害的预防和治理水平。

  5. 农村产权交易对象征信：大数据应用于农村产权交易对象征信，通过数据分析和挖掘，为农村产权交易提供信息支持和信用评估。

总结：

这本出版物的内容确实为对大数据感兴趣或初入门的读者提供了极具价值的信息。它不仅涵盖了大数据的基本概念和系统架构，还介绍了大数据的发展历史和未来的应用趋势，从而激发了读者对大数据行业的信心和兴趣。

然而，在阅读过程中，读者可能会遇到一些问题，其中之一是关于大数据与小数据的对比。虽然书中提到了小数据是大数据的微观观察，并且是大数据计算机未发展的前身，但对于小数据的具体含义和来源还不够清晰。在介绍小数据概念之前，可以通过举例来解释，例如提到维基百科中的案例：小数据可以是个体自身的量化，如运动手环和智慧手表收集个人的身体信息。这样的例子更具体地说明了小数据的概念和应用。

另一个问题是关于区块链技术的解释。书中多次提到区块链的特点，如安全、透明、可查，但是却没有详细说明区块链是如何实现这些特点的。读者可能会想知道区块链是如何为各个行业赋能的，是否可以将其理解为一种加密技术，或者是一个全新的行业或逻辑体系。通过更详细的解释和案例，读者可以更好地理解区块链技术的本质和应用场景。

最后，在结构上，这本出版物确实很有吸引力。它从大数据的过去到未来进行了全面的探讨，涵盖了从军事领域到医疗研发再到社区治安的各个领域。这种结构使读者能够清晰地了解大数据在不同领域的应用，并对未来充满希望。

综上所述，通过对小数据和区块链技术的更详细解释和举例，以及对大数据应用领域的更深入探讨，这本出版物可以帮助读者更全面地理解大数据的概念和意义，从而更好地应用于实际生活和工作中。
